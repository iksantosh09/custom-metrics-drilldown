The Cuanto Java API provides a programmatic interface for creating, updating and querying for Cuanto test objects. It allows you to submit test results to the Cuanto server and retrieve results from the Cuanto server. The primary point of usage is the {@link cuanto.api.CuantoConnector}.

Here are some examples:

		CuantoConnector cuanto = CuantoConnector.newInstance("http://mycuanto.url", "CuantoProjectKey");

		// create a TestRun

		TestRun testRun = new TestRun(new Date());
		testRun.addTestProperty("Milestone", ".92");
		testRun.addTestProperty("Changelist", "34523");
		testRun.addTestProperty("Hostname", "testrunner");

		// add the TestRun to Cuanto
		cuanto.addTestRun(testRun);

		// create a TestOutcome
		TestOutcome outcome = TestOutcome.newInstance("com.mytestpackage.sample", "testMySample", TestResult.Fail);
		outcome.setDuration(12503L);

		// add the TestOutcome to the TestRun and Cuanto
		cuanto.addTestOutcome(outcome, testRun);

		// fetch all TestOutcomes associated with this TestRun
		List<TestOutcome> testOutcomes = cuanto.getTestOutcomesForTestRun(testRun, 0, 100, TestOutcome.Sort.FULL_NAME, "asc")
